# Tarea

Implementar la función flip

## Argumentos

* función con dos argumentos

## Conditions

* retorna una función que aplica los argumentos en orden inverso

## ejemplo

```javascript
const concat = (a, b) => `${a}-${b}`
const concatReverse = flip(concat)
concatReverse("a", "b") // => 'b-a'
```
